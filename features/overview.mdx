## Design philosophy

Arda embraces a **medieval manuscript aesthetic** with:

- **OLED-friendly true black** backgrounds for comfortable reading
- **Warm parchment tones** (#ddd7be, #c9c2a5) for text
- **Burgundy accents** (#8b2d3e) throughout the interface
- **Ornamental flourishes** using Unicode characters (◊, ⚔, ❉)
- **Smooth animations** with carefully tuned timing curves

## Color modes

Arda is designed exclusively for **dark mode**. The theme does not include a light mode variant.
